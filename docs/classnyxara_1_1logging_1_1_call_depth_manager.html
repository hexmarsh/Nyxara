Detailed Description

Manages logging depth based on call depth (per thread).

Used internally by the logging system to add call depth information to logs for nested function calls. This helps track call hierarchy in complex systems. Call depth is tracked per-thread.

CallDepthManager is used automatically by the FunctionTracer class, but can also be manually enabled or disabled via SetEnabled().
