\doxysubsection{Detailed Description}
Core loggign utility for the Nyxara engine. 

Provides a static interface for managing log categories, verbosity levels, and logging messages with optional call-\/depth indentation.

The logging system can be explicitly initialized by calling \doxylink{class_nyxara_1_1_logging_1_1_logger_a1a30ca88ffda7150e4051213195d207b}{Init()}, but it also supports lazy initialization. Loggers will automatically initalize themselves the first time they are used, even if \doxylink{class_nyxara_1_1_logging_1_1_logger_a1a30ca88ffda7150e4051213195d207b}{Init()} is not called manually.
