\doxysubsection{Detailed Description}
Manages logging indentation based on call depth (per thread). 

Used internally by the logging system to visually indent logs for nested function calls. This helps track call hierarchy in complex systems. Call depth is tracked per-\/thread.

\doxylink{class_nyxara_1_1_logging_1_1_call_depth_manager}{Call\+Depth\+Manager} is used automatically by the \doxylink{class_nyxara_1_1_logging_1_1_function_tracer}{Function\+Tracer} class, but can also be manually enabled or disabled via \doxylink{class_nyxara_1_1_logging_1_1_call_depth_manager_ab09ddf71989ee8a79cc4a8355cf4eb30}{Set\+Enabled()}.
